# IntegraXL
Roland INTEGRA-7 Editor &amp; Librarian

This is a work in progress as seen in the screenshot and far from finished. The [Integra](https://github.com/X-Lars/IntegraXL/tree/master/Integra) 
project provides the all INTEGRA-7 interaction and data storage which should be reusable in other .NET projects. The [IntegraControls](https://github.com/X-Lars/IntegraXL/tree/master/IntegraControls) contains the custom
build INTEGRA-7 controls, like the step filter shown in the preview.

References used
- [ToolsXL](https://github.com/X-Lars/ToolsXL) For storing user configuration.
- [MidiXL](https://github.com/X-Lars/MidiXL) For MIDI interaction.
- [ControlsXL](https://github.com/X-Lars/ControlsXL) Application layout and custom controls.
- [StylesXL](https://github.com/X-Lars/StylesXL) For dynamic skinning the application and controls.
